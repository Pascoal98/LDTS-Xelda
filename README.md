# XELDA

A mock version of the old style Zelda games.

---

### Controls 

- **Arrow-up** : Moves the character up.
- **Arrow-down** : Moves the character down.
- **Arrow-left** : Moves the character left.
- **Arrow-right** : Moves the character right.
- **Q** : Exits game.
